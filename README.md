# 🎮 Tower of Hanoi Game

An interactive web-based implementation of the classic Tower of Hanoi puzzle built using HTML, CSS, and JavaScript.

---

## 🚀 Features
- Drag and drop disk movement
- Real-time move counter
- Timer tracking
- Minimum move calculation (2^n - 1)
- Auto-solve feature using recursion
- Responsive and clean UI

---

## 🧠 Game Rules
- Only one disk can be moved at a time
- Only the top disk can be moved
- A larger disk cannot be placed on a smaller disk

---

## ⚙️ Technologies Used
- HTML
- CSS
- JavaScript
